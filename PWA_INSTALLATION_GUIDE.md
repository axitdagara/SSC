# PWA Installation Guide - SSC (Sculpt Soft Cricketers)

## What is a PWA?

A **Progressive Web App (PWA)** is a web application that can be installed on your device like a native app. It provides:
- ✅ Fast loading times
- ✅ Offline functionality
- ✅ Push notifications support
- ✅ Home screen icon
- ✅ Full-screen experience (no browser bar)

---

## How to Install SSC App

### **On Android (Chrome, Edge, Opera)**
1. **Look for the install banner** at the bottom of the screen
   - Or tap the menu icon (⋮) in the top-right corner
   
2. **Tap "Install" or "Install app"**
   - Confirm any prompts
   
3. **The app will be added to your home screen**
   - You can now open it like any other app

### **On iOS (Safari)**
1. **Tap the Share button** (square with arrow ↗️)
   
2. **Scroll down and tap "Add to Home Screen"**
   
3. **Enter a name** (or keep "SSC")
   
4. **Tap "Add"** to confirm
   
5. **The app will appear on your home screen**
   - Open it anytime like a regular app

### **On Desktop/Windows (Chrome, Edge)**
1. **Look for the install icon** in the address bar (usually on the right side)
   - Or use the menu → "Install SSC"
   
2. **Click the icon to install**
   
3. **Confirm the installation**
   
4. **The app will launch in a window** without the browser bar

### **On macOS (Safari, Chrome)**
1. **Same as Desktop Windows steps above**
   - Or use Safari menu → File → "Add to Dock" (if available)

## Troubleshooting

### **"Install" option not showing?**
- **Android**: Make sure you're using Chrome, Edge, or Opera
- **iOS**: Must use Safari; Chrome doesn't support PWA installation on iOS
- **Desktop**: Try using Chrome or Edge instead of Firefox

### **App not working offline?**
- Some features require internet (live scoring, sync)
- Basic match viewing should work offline
- Reload the app to force cache update

### **Want to remove the app?**
- **Android**: Long-press the app icon → "Uninstall"
- **iOS**: Long-press the app icon → "Remove App" → "Remove from Home Screen"
- **Desktop**: Remove like any other application from your system

## How PWA Works

The app uses:
- **Service Worker**: Caches app files for faster loading
- **Manifest**: Defines app metadata (name, icons, colors)
- **HTTPS**: Secure connection for safe installation

All your data is stored locally in your browser and synced with our servers when online.

## Questions?

For technical issues:
1. **Clear app cache**: Settings → Apps → SSC → Storage → Clear Cache
2. **Reinstall**: Uninstall and install again from the address bar
3. **Update**: The app auto-updates in the background
4. **Fallback**: Visit the web version at https://ssc.app
